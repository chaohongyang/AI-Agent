# Argus — Social Media Intelligence

A modern, single-page landing page for **Argus**, a social media intelligence platform. Built with pure HTML and CSS — no frameworks, no dependencies.

## Overview

Argus is a fictional SaaS product that provides real-time social media monitoring, sentiment analysis, and trend detection across major platforms.

## Features

The landing page includes:

- **Hero section** — headline, subtext, and CTA buttons
- **Features section** — platform capabilities (real-time monitoring, sentiment analysis, trend detection, etc.)
- **Pricing section** — tiered pricing cards
- **FAQ section** — accordion-style frequently asked questions
- **Footer** — navigation and branding

## Tech Stack

- HTML5
- CSS3 (custom properties, glassmorphism, animations)
- Vanilla JavaScript (FAQ accordion, scroll reveal)
- Google Fonts (Inter)

## Getting Started

No build step required. Just open `index.html` in a browser:

```bash
open index.html
```

Or serve locally:

```bash
npx serve .
```
